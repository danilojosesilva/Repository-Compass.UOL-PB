#### **I - Segurança em Aplicações WEB:**

#### **Resumo:**

<details><summary></summary> <p> O curso "Segurança em Aplicações WEB" é um programa abrangente projetado para capacitar os participantes a desenvolver aplicações web com foco na segurança desde a concepção do projeto. O instrutor, Samuel Gonçalves Pereira, é um profissional experiente em Redes e Segurança, com vasta experiência na administração de sistemas de segurança em redes de grande porte.

O curso aborda uma ampla gama de tópicos essenciais relacionados à segurança em aplicações web. Começando pelos fundamentos, os participantes são introduzidos à arquitetura e tecnologia das aplicações web atuais, bem como às metodologias de teste de invasão, com destaque para o uso do OWASP e OWASP ZAP como ferramentas de pentest. O laboratório para o curso é estruturado de forma a fornecer um ambiente prático para a aplicação dos conhecimentos adquiridos.

Um dos principais focos do curso é a abordagem das principais vulnerabilidades encontradas em aplicações web. Isso inclui uma exploração detalhada das técnicas de injeção de código malicioso, como injeção de SQL, injeção de comandos remotos, Cross-Site Scripting (XSS) e CSRF. Os participantes aprendem a identificar essas vulnerabilidades e a aplicar as melhores práticas para corrigi-las e proteger as aplicações.

Outro aspecto fundamental abordado no curso é a criptografia e o transporte seguro de credenciais. Os participantes aprendem sobre os riscos associados ao transporte de credenciais por canais inseguros, a importância da implementação de políticas de senhas fortes, as falhas comuns no mecanismo de autenticação e as técnicas de teste dos mecanismos de autorização e criptografia.

O curso também explora técnicas de escalonamento de privilégios, como o uso de sniffers, ferramentas de força bruta e testes de penetração. Os participantes aprendem a identificar e mitigar essas ameaças, bem como a fortalecer a segurança das aplicações em diferentes níveis.

Por fim, o curso aborda estratégias de defesa e mitigação, incluindo o uso de proxies de aplicação, SSL, HTTPS e a implementação de práticas de segurança no próprio código das aplicações.

Cada módulo do curso é acompanhado por exercícios práticos e laboratórios, permitindo aos participantes aplicar os conceitos aprendidos em cenários reais.
</p>

#### **I - Programação funcional: O que é?:**

#### **Resumo:**

<details><summary></summary> <p> A programação funcional é um paradigma de programação que se baseia no uso de funções puras e na imutabilidade dos dados. Ao contrário da programação imperativa, que se concentra em como realizar tarefas específicas, a programação funcional se concentra em expressar lógica e comportamento através de funções.

Nesse estilo de programação, as funções são tratadas como cidadãs de primeira classe, o que significa que podem ser atribuídas a variáveis, passadas como argumentos para outras funções e retornadas como resultados de outras funções. Essa abordagem permite a criação de funções mais flexíveis, reutilizáveis e modularizadas.

Um dos principais conceitos da programação funcional é a imutabilidade dos dados, o que significa que os valores não são alterados após serem criados. Em vez disso, as funções criam novos valores a partir dos dados existentes, mantendo a integridade dos dados originais. Isso evita efeitos colaterais indesejados e facilita o raciocínio sobre o código.

A programação funcional também se baseia fortemente no uso de funções de ordem superior, que são funções que podem receber outras funções como argumentos ou retorná-las como resultado. Isso permite a criação de abstrações poderosas e a composição de funções, o que facilita a construção de programas complexos a partir de partes menores e mais simples.

Outro conceito importante na programação funcional é a recursão, que é a capacidade de uma função chamar a si mesma para resolver um problema de forma iterativa. A recursão é frequentemente usada para lidar com estruturas de dados recursivas, como listas ou árvores.

A programação funcional oferece uma série de benefícios, como código mais conciso e legível, maior facilidade de teste e depuração, maior reutilização de código e a possibilidade de aproveitar técnicas avançadas, como memoização e programação assíncrona.

Em resumo, a programação funcional é um paradigma de programação que se baseia no uso de funções puras e imutabilidade dos dados. Ela oferece uma abordagem flexível e poderosa para resolver problemas complexos, facilitando a construção de código modular, reutilizável e de fácil compreensão.
</p>
</details>

<details><summary></summary> <p> O curso "Introdução Completa ao AWS QuickSight [Edição 2023]" é um programa abrangente que oferece aos participantes a oportunidade de adquirir conhecimentos sólidos sobre o Amazon QuickSight. Durante o curso, os alunos exploram uma variedade de tópicos essenciais relacionados ao uso eficaz dessa poderosa ferramenta de análise de dados.

O curso ensina sobre o fluxo de uso do QuickSight, desde o processo de carregamento e preparação dos dados até a criação de visualizações atraentes e informativas. Eles descobrem como importar e organizar dados de diferentes fontes, como bancos de dados e planilhas, e aprender a transformar e limpar esses dados para análise.

Em seguida, o curso aborda a criação de visualizações impactantes no QuickSight. Os participantes aprendem a criar gráficos, tabelas e painéis interativos que permitem explorar e analisar os dados em detalhes. Eles também exploram recursos avançados, como cálculos personalizados e a criação de indicadores-chave de desempenho (KPIs) para monitorar métricas importantes.

Além disso, o curso aborda a aplicação de filtros e controles para refinar as visualizações e fornecer uma análise mais específica. Os participantes aprendem a filtrar os dados com base em critérios específicos e a criar controles interativos para os usuários finais personalizarem suas próprias análises.

Uma parte fundamental do curso é dedicada à criação de análises de tendências e previsões. O curso ensina a identificar padrões nos dados e a criar gráficos e modelos que ajudam na previsão de tendências futuras. Essas habilidades são valiosas para a tomada de decisões estratégicas e o planejamento de negócios.

Por fim, o curso explora o compartilhamento de relatórios e dashboards criados no QuickSight. O curso ensina a publicar e compartilhar análises com outras pessoas, definindo permissões de acesso e permitindo que os usuários finais interajam com os dados por meio de filtros e controles.
</p>
</details>
